How to Use Code Visualizer

🎯 Getting Started

Paste JavaScript/React code or enter a project folder path to visualize code structure as an interactive graph.

🖱️ Interactions

  • Click a node to see details and highlight connections
  • Drag nodes to rearrange the layout
  • Scroll to zoom in/out
  • Click + Drag the background to pan

🎨 Node Types

  • Files — source code files
  • Functions — declared functions
  • Components — React components
  • Events — event handlers/listeners
  • Routes — API/page routes
  • API Calls — HTTP requests
  • Hooks — React hooks
  • Imports — module imports

⌨️ Keyboard Shortcuts

  • Ctrl + Enter — Visualize pasted code
  • Escape — Clear selection / Close panels
  • F — Fit graph to view
  • R — Reset highlights
