## 🚀 Overview

The **Vibe Code Visualizer** allows you to instantly map out the architecture of your React and JavaScript projects. Simply point the tool to any folder, and it will dynamically parse the Abstract Syntax Tree (AST) to generate an interactive, beautiful graph of your code structure.

Designed with a premium "Slate/Glassmorphism" aesthetic inspired by modern enterprise docs, the dashboard provides a clean, breathable space for analyzing complex logic and dependencies.

### 🌟 Features

*   **📂 Clustered File Layout**: Cleanly groups functions, components, and imports into precise rigid containers. See exactly what lives inside each file without messy overlap.
*   **🔭 Dynamic Force Physics**: A spaced-out, spring-based visualization mode with tight, straight dependency edges. Drag and untangle specific nodes to see how they connect globally.
*   **🔍 AI-Powered Purpose Tags**: An built-in heuristic engine automatically deduces the *purpose* of specific functions, hooks, and classes, instantly clarifying what the code does within the visualization interface.
*   **🌌 Premium Glassmorphic UI**: High-end deeply saturated Navy (`Slate-950`) backgrounds, brilliant Electric Blue accents, and sharp `Space Grotesk` typography built entirely without external styling frameworks.

## 🛠️ Getting Started

### Prerequisites

You need [Node.js](https://nodejs.org/) installed on your machine.

### Installation & Run

1. Clone this repository locally:
   ```bash
   git clone https://github.com/JalAnghan/vide-coder-tool.git
   ```
2. Navigate into the folder:
   ```bash
   cd vide-coder-tool
   ```
3. Install the minimal dependencies (Express & Babel AST Parser):
   ```bash
   npm install
   ```
4. Start the backend parser server and front-end simultaneously:
   ```bash
   node server.js
   ```
5. Open your browser to `http://localhost:3000`

### Visualize Your Own Project 

1. On the left sidebar, click the **"Open Folder"** tab.
2. Enter the **absolute path** to your own local Node/React project (e.g., `C:\Path\To\YourProject\src`).
3. Hit **Parse**, and interact with the instant architectural graph!
